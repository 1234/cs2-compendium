# Refrag Routines

[Refrag](https://play.refrag.gg) is an in-game CS2 training platform. Unlike KovaaK,
which runs in a standalone aim trainer, Refrag runs inside CS2 itself — same movement,
same weapon feel, same physics. The gap between trainer and game is much smaller.

These routines run directly in-game using Refrag's bot system. No separate app required.

---

## The four routines

### Warm-up
**Link:** https://play.refrag.gg/routines/ee66b8476febf3d2e500

Low-intensity activation. The goal is not to grind — it's to get your hands moving
before you go into anything demanding. Used on Saturdays (scheduled low-intensity day)
and as a standalone if you're short on time.

**What it trains:** Re-activating movement and click timing at low stakes. Not a skill-
building session — purely a warmup vehicle.

**When to use it:** Saturday in the weekly rotation, or any day you want a lighter session.

---

### Aim
**Link:** https://play.refrag.gg/routines/936d3034ad09b84cc69e

The core aim routine. Covers clicking, target switching and entry scenarios using
CS2 bots at realistic distances and angles. Because it's in-engine, it trains you
against the exact hitboxes and movement patterns you'll see in real matches.

**What it trains:** Click timing, target acquisition, head-level crosshair placement,
transitioning between multiple targets. The competitive-specific version of what
clicking scenarios in KovaaK target.

**When to use it:** Monday, Wednesday, Friday in the weekly rotation.

---

### Spray
**Link:** https://play.refrag.gg/routines/ca20c447c5b1a5f51e9f

Dedicated spray control practice with CS2 weapons. Spray patterns in CS2 are
deterministic — same pattern every time — but controlling them under pressure is
a separate motor skill from knowing the pattern.

**What it trains:** Recoil control, spray-to-kill at medium range, the muscle memory
of pulling through the full spray pattern while staying on target.

**When to use it:** Tuesday (full spray) in the weekly rotation.

---

### Spray (easier)
**Link:** https://play.refrag.gg/routines/14b37a68e1c2f23388b8

A reduced-intensity version of the spray routine. Same mechanics, lower volume and
difficulty. Exists because spray training is physically demanding — by Thursday
fatigue has accumulated from the week and grinding the full routine against a tired
body produces bad habits.

**What it trains:** Same spray fundamentals, at a pace appropriate for mid-week fatigue.

**When to use it:** Thursday in the weekly rotation — always the easier version, not the full one.

---

## Why in-game training matters alongside KovaaK

KovaaK is valuable for isolating specific mechanics (micros, tracking, target switching)
in a controlled environment. But it's a different game — different physics, different
hitboxes, different movement feel.

Refrag bridges the gap. It trains the same skills but inside CS2, which means:
- No translation layer between aim trainer and game
- Bot movement matches what real players do
- Weapon recoil is the actual CS2 model, not a simulation

**The rule of thumb from HaiX's research:** Use KovaaK to build the skill, use Refrag
to transfer it to the game. Running both is more effective than running either alone.
